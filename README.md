# Machine Learning Model Comparison

## Objective
The aim of this project is to compare multiple machine learning models on a dataset and evaluate their performance.

## Methods
- Data preprocessing and cleaning
- Feature engineering
- Model training using:
  - k-Nearest Neighbours (kNN)
  - Decision Tree
  - Support Vector Machine (SVM)
- Model evaluation using:
  - Accuracy
  - Precision
  - Recall
  - F1-score
- Cross-validation for robust performance assessment

## Results
The models were compared based on evaluation metrics, highlighting differences in performance depending on the algorithm used.

## Key Insights
- Different machine learning models perform differently depending on the dataset
- Evaluation metrics provide deeper insight than accuracy alone
- Feature engineering improves model performance

## Tools Used
Python, pandas, scikit-learn, matplotlib, seaborn

## Note
This project was developed as part of MSc coursework and refined for portfolio purposes to demonstrate machine learning workflows and model evaluation.
